# ChatGPT Poster Generation Prompt

Copy and paste this entire prompt into ChatGPT:

---

Create a professional, modern poster for "StockPredict AI" - an AI-powered stock market prediction platform. The poster should be suitable for academic presentations, portfolio showcases, or project demonstrations.

## Poster Specifications:

**Dimensions**: 16:9 aspect ratio (1920x1080px or similar), suitable for digital display or print

**Design Style**: 
- Modern, professional financial technology aesthetic
- Dark theme with vibrant accent colors (blue, purple, green gradients)
- Clean, minimalist layout with clear information hierarchy
- Futuristic AI/tech vibe with subtle financial chart patterns or data visualization elements

## Content to Include:

### Header Section:
- **Main Title**: "StockPredict AI" (large, bold, gradient text - blue to purple)
- **Tagline**: "AI-Powered Stock Market Analysis & Prediction Platform"
- **Subtitle**: "Advanced Machine Learning Ensemble for Intelligent Trading Decisions"

### Main Features Section (Visual Icons + Text):
1. **Multi-Model AI Ensemble**
   - 6 Machine Learning Models: Linear Regression, LSTM Neural Networks, ARIMA Time Series, XGBoost, Decision Tree, SVM
   - Icon: Brain/Neural Network visualization

2. **Advanced Charting & Analysis**
   - Professional TradingView-style candlestick charts
   - Real-time price tracking across 7 timeframes (5m, 15m, 1H, 4H, 1D, 1W, 1M)
   - Technical indicators: RSI, MACD, Moving Averages, Bollinger Bands
   - Icon: Line chart with candlesticks

3. **Pattern Recognition**
   - Automatic candlestick pattern detection (Doji, Hammer, Engulfing, etc.)
   - Strength scoring and signal quality assessment
   - Icon: Target/Bullseye

4. **Backtesting Engine**
   - Walk-forward analysis
   - Performance metrics: Sharpe Ratio, Win Rate, Max Drawdown
   - Equity curve visualization
   - Icon: Test tube/Analytics

5. **S&P 500 Coverage**
   - Access to 500+ stocks
   - Intelligent search and filtering
   - Icon: Database/Server

6. **Real-Time Market Data**
   - Live data from Yahoo Finance
   - Automatic updates and price quotes
   - Icon: Lightning bolt/Refresh

### Technology Stack Section:
**Backend & AI**: Python, Flask, TensorFlow, Keras, XGBoost, scikit-learn, statsmodels, ARIMA, LSTM

**Frontend**: React, TypeScript, Vite, Tailwind CSS, Lightweight Charts, Radix UI

### Key Statistics/Metrics:
- **6** ML Models Combined
- **500+** S&P 500 Stocks
- **7** Timeframes Supported
- **10+** Candlestick Patterns Detected

### Visual Elements:
- Include a mockup or visualization of:
  - Stock price chart with prediction overlay (show historical data + future predictions)
  - Model performance comparison chart
  - Technical indicators dashboard
  - Pattern detection markers on chart

### Color Scheme:
- Primary: Deep blue (#1e3a8a) to purple (#7c3aed) gradient
- Accent: Green (#10b981) for positive metrics, Orange (#f59e0b) for warnings
- Background: Dark navy/black (#0f172a or #1e293b)
- Text: White/Light gray for contrast
- Charts: Use vibrant colors (blue, green, red, purple) for data visualization

### Typography:
- Main title: Bold, large, sans-serif (like Inter or Poppins)
- Headings: Semi-bold, medium-large
- Body text: Regular weight, readable size
- Numbers/metrics: Monospace font for technical feel

### Layout:
- Split layout: Left side with features/icons, right side with chart visualizations
- OR: Top section with title and tagline, middle section with features grid, bottom section with tech stack and stats
- Include QR code placeholder (optional, bottom corner)
- Footer: "Built with ❤️ using cutting-edge AI and modern web technologies"

### Additional Design Elements:
- Subtle geometric patterns or grid lines in background
- Small data visualization snippets (mini charts, graphs)
- Glowing effects around key elements (AI brain icon, prediction lines)
- Professional, trustworthy aesthetic similar to Bloomberg Terminal or TradingView

### Disclaimers/Small Print (if space allows, bottom):
"Educational purposes only. Past performance does not guarantee future results."

---

**Make it visually striking, professional, and suitable for showcasing a sophisticated AI-powered financial technology platform. The poster should immediately communicate the advanced nature of the system while remaining accessible and impressive.**
